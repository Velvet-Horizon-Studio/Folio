Features
🖼️

Multi-folder Slideshow

Load images from multiple folders at once and browse them as a smooth slideshow with transition effects.

⊞

Thumbnail Browser

Resizable grid view with lazy loading, folder grouping, and instant jump to any image.

🔍

Zoom & Pan

Scroll or use arrow keys to zoom from 25 % to 500 %. Drag to pan. Right-click resets to 100 %.

⚡

Live Folder Watch

Thumbnails refresh automatically when images are added or removed from watched folders.

🤖

AI Image Metadata

Reads Stable Diffusion generation parameters embedded in PNG files — prompt, negative prompt, sampler, steps, and more.

✏️

File Management

Rename, delete, convert to PNG/JPEG, copy, and move files — individually or in bulk via the action bar.

📂

Folder Management

Drag the ⠿ handle to reorder folders. Right-click a folder to move all its images to another folder in one step.

↔️

Move Images

Right-click any thumbnail to move it to another folder. Select multiple and use the bulk action bar to move them all at once.

⚙️

Startup Options

Choose whether Folio resumes your last image, starts from the first, or jumps to the last image each time it opens.

💾

Persistent Settings

Remembers your last folder, image position, sidebar width, and thumbnail size between sessions.

🪟

Fullscreen Mode

Double-click any image or press F11 to go fullscreen. Controls auto-hide after a few seconds.
